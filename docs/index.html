DL Link Plugin Documentation

Overview Installation Usage API Reference

Overview

The DL Link plugin is a device driver for N.I.N.A. that allows you to control Digital Logger outlets. It provides functionality to turn-on, tunr-off and cycle outlets in the switch equipment section as well as in sequences..

Installation

To install the DL Link plugin:

  1. Download the plugin from the official repository.
  2. Or place the plugin files manually in the N.I.N.A. plugins directory.
  3. Restart N.I.N.A. to load the plugin.

Usage

To use the DL Link plugin:

  1. Navigate to the equipment settings in N.I.N.A.
  2. Select the DL Link plugin from the list of installed plugins.
  3. Configure the server address, username, and password in the plugin settings. Note that the password will not be ecnrypted, as DL switches use digest auth over http.
  4. Connect to the device and manage outlets as needed.
